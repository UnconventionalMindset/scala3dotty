## Sample project using Scala 3 compiler

### Usage

This is a normal sbt project, you can compile it with
```
sbt compile
```
and run it with
```
sbt run
```
to start a Scala 3 REPL, use:
```
sbt console
``` 
and finally, to run tests:
```
sbt test
```
### Tooling
Scala Version: `3.2.1-RC1-bin-20220716-bb9c8ff-NIGHTLY`

SBT `1.6.2`

Tested using JVM `17`
